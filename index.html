Welcome to My Personal Website

Home | About | Resume

Hello, and welcome to my personal website. This page was created as part of a web development assignment, and it is designed to show the basics of HTML, CSS, and JavaScript in one place. I am interested in learning how websites are built because the internet plays a major role in how people communicate, work, and share information today. Even a simple webpage can be useful when it is organized well and easy to navigate.

Building a website also teaches valuable problem-solving skills. Every page needs a clear structure, readable text, and working links so users can move through the site without confusion. In this project, I wanted to create something that looks clean and professional while still meeting the technical requirements. Learning these basics is important because web design is a very practical skill that can be applied in school, business, and personal projects.

This site includes an about page, a resume page, a shared CSS file, and an interactive JavaScript game. Together, these parts show how multiple files can work together to create a complete website. I also wanted the content to be easy to read, since design should support the user rather than distract from the information. A simple layout with clear headings can make a website feel more professional and more accessible to visitors.

In addition to the written content, this homepage includes a Tic-Tac-Toe game written in JavaScript. Small interactive features like this make websites more engaging and show that web pages can do more than just display text. As I continue learning, I hope to build larger and more advanced projects that combine structure, style, and interactivity in an effective way.
